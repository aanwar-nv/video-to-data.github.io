# Human Demos to Robot Data @ CoRL 2026

Website for the CoRL 2026 Workshop **Learning from Human Demonstrations: Video-to-Data, Reconstruction, and Robot Grounding**.

Content is driven by `assets/js/config.js` (speakers, organizers, schedule). Drop headshots into `assets/images/` and update the placeholder image paths in `config.js`.

Template adapted from [eval-deploy.github.io](https://github.com/AbrarAnwar/eval-deploy.github.io).
